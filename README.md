Global Terrorism Analysis (1970 - 2017)


📌 Project Overview
This project performs an Exploratory Data Analysis (EDA) on the Global Terrorism Database (GTD). The goal is to identify patterns in global security breaches, regional instability, and tactical shifts over nearly five decades. By transforming raw data into visual insights, this analysis highlights the human and geopolitical impact of terrorism.



🔍 Research Focus
Unlike standard data cleaning tasks, this analysis focuses on:

Temporal Trends: Identifying "peak years" and correlating them with global geopolitical shifts.

Tactical Analysis: Evaluating the most common attack methods (Bombings vs. Armed Assaults).

Geospatial Hotspots: Mapping the migration of conflict zones across continents.

Casualty Impact: Analyzing the severity of attacks based on target types.



🛠️ Tech Stack
Language: Python

Environment: Google Colab / Jupyter Notebook

Data Analysis: Pandas, NumPy

Visualization: Seaborn, Matplotlib, Plotly




📊 Key Findings

Historical Surge: A significant increase in recorded incidents post-2010.

Regional Dominance: Detailed breakdown of why specific regions (South Asia, Middle East) appear more frequently in the data.

Weaponry: Correlation between weapon availability and attack frequency.
